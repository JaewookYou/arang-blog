## 문제 상황

셀레니움 봇 만드려는데 그냥 `pip3 install selenium` 하고 기존 쓰던 코드 적용하니 온갖 에러가 났다.

---

## 에러 1: LATEST_RELEASE not found

webdriver-manager 적용 시 최신 크롬 드라이버를 못 찾아서 에러가 발생.

### 해결

```bash
pip3 install git+https://github.com/SergeyPirogov/webdriver_manager
```

최신 webdriver_manager를 GitHub에서 직접 설치하면 해결된다.

---

## 에러 2: Python 버전 문제

webdriver-manager 최신 버전은 **Python 3.7 이상**이 필요하다.

### 해결

```bash
# Python 3.7 이상 버전 설치
pyenv install 3.10.0
pyenv global 3.10.0
```

---

## 에러 3: TypeError: __init__() got unexpected keyword argument

```
TypeError: __init__() got multiple values for argument 'options'
```

실제론 `options` 때문에 에러나는 건 아니고, Selenium v4.10+ 에서 API가 변경되었다.

### 해결

**기존 코드 (v4.9 이하):**

```python
from selenium import webdriver
from webdriver_manager.chrome import ChromeDriverManager

driver = webdriver.Chrome(ChromeDriverManager().install())
```

**수정된 코드 (v4.10+):**

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager

service = Service(ChromeDriverManager().install())
driver = webdriver.Chrome(service=service)
```

핵심은 `Service` 객체를 사용하는 것이다!

---

## 전체 예제 코드

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.chrome.options import Options
from webdriver_manager.chrome import ChromeDriverManager

# Chrome 옵션 설정
options = Options()
options.add_argument('--headless')  # 헤드리스 모드
options.add_argument('--no-sandbox')
options.add_argument('--disable-dev-shm-usage')

# Service 객체로 드라이버 설정
service = Service(ChromeDriverManager().install())

# WebDriver 실행
driver = webdriver.Chrome(service=service, options=options)

# 페이지 접속
driver.get('https://example.com')
print(driver.title)

driver.quit()
```
